# sham_coil_model
3D model of a 70mm figure-of-8 sham coil. The current flows in the same direction in both wings, creating destructive magnetic interference at the center of the coil. Can be used directly with SimNIBS software for computational modeling. 

Since no sham coil model is available in the software, we adapted the coil model by Thielscher et al.* to reflect the reversed current directions in the sham coil, by flipping the magnetic dipoles over the z-axis in one coil wing, effectively reversing the field direction in one wing to match the sham configuration. 

* Thielscher, A. and T. Kammer, Electric field properties of two commercial figure-8 coils in TMS: calculation of focality and efficiency. Clin Neurophysiol, 2004. 115(7): p. 1697–708.
